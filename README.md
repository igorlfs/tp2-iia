Standard Python

## Run

Run with `python qlearning [path_to_map_file] [id_modification] Xi Yi N`[^1], e.g.

```sh
python qlearning maps/mapa_teste.map standard 0 3 100000
```

[^1]: Where `[id_modification]` is either `standard`, `positive`, `stochastic`.
